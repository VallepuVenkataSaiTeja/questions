## 1. Prerequisites (must know first)

Before React, get comfortable with:

* HTML fundamentals (forms, semantic tags)
* CSS basics (flexbox, grid, positioning)
* JavaScript basics:

  * variables, functions, arrays, objects
  * loops, conditions
  * ES6+ features (very important)

### ES6+ topics to focus on:

* arrow functions
* destructuring
* spread/rest operators
* map/filter/reduce
* modules (import/export)
* promises + async/await
